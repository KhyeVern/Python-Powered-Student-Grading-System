# Student Grading System (Python)

## Project Description
The goal of this project is to develop a Student Grading System that manages student records and their assessment results. The system is designed to process and organize student data efficiently while providing meaningful insights into academic performance.

The program reads student information and assessment data from external text files, calculates overall grades, assigns appropriate letter grades, and displays class performance summaries.

## Data Sources
The system uses the following text files as input:

- **students.txt** – Contains student ID, name, and email  
- **grades.txt** – Contains student ID, course name, marks, and grade  
- **courses.txt** – Contains course ID and course name  

## Features
- Load and manage student records from files  
- Process and link student, course, and grade data  
- Calculate overall grades for each student  
- Assign letter grades based on performance  
- Display class performance summaries  

## Purpose
This project demonstrates the use of file handling, data processing, and basic program logic in Python to build a functional academic management system.
